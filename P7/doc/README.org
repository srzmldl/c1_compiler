* 注意事项以及检查方法
  - 在跟目录下 `bash run.sh`
  - 看到精心设计的test的输出结果.
    + testGreat.c1 测试了各种基本语句,所有语句都包含有了测试. 输出如果是100递减的一串数表明正确.
    + factorial.c 测试了递归函数调用,全局变量和局部变量的引用

  - make clean 删除不必要的内容
